# USE CASES for Starter Exercise

### Use Case 01: Initialize Animal Collection

**As a system user**
</br>**I want** the application to start with a pre-populated list of animals
</br>**So that** I can see example data when the program runs

### Use Case 02: Display Current Animal Inventory

**As a user**
</br>**I want** to see all animals currently in the collection
</br>**So that** I know what animals exist in the system

### Use Case 03: Add New Animal to Collection

**As a user**
</br>**I want** to add a new animal to the collection
</br>**So that** I can expand the inventory with additional animals

### Use Case 04: Track Animal Count with Dual Methods

**As a developer**
</br>**I want** to maintain both a manual counter and use the list length
