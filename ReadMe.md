# Use Cases for Enhanced Animal Collection System 

### Use Case 01: Display Interactive Menu System

**As a user**
</br>**I want** to see a menu with multiple options
</br>**So that** I can choose which action to perform

### Use Case 02: Implement Continuous Operation Loop

**As a user**
</br>**I want** the application to continue running after each action
</br>**So that** I can perform multiple operations without restarting

### Use Case 03: List Animals with Numbered Display

**As a user**
</br>**I want** to view all animals with sequential numbering
</br>**So that** I can easily reference specific animals by number

### Use Case 04: Add Animal with Menu Integration

**As a user**
</br>**I want** to add animals through a menu option
</br>**So that** I can add multiple animals in one session

### Use Case 05: Validate Animal Input Length

**As a system**
</br>**I want** to ensure animal details meet minimum length requirements
</br>**So that** incomplete or invalid data is not accepted

### Use Case 06: Exit Application Gracefully

**As a user**
</br>**I want** to exit the application cleanly
</br>**So that** I can end my session when finished

### Use Case 07: Handle Invalid Menu Selections

**As a system**
</br>**I want** to inform users when they make invalid menu choices
</br>**So that** they understand their input was not recognized
