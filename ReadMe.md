# Use Cases for further enhancements with collections

### Use Case 01: Parse and Display Animal Details in Structured Format

**As a user**</br>
**I want** to see animal details in a readable, labeled format</br>
**So that** I can easily understand each animal's attributes

### Use Case 02: Collect Animal Data Through Individual Field Prompts

**As a user**
</br>**I want** to enter each animal attribute separately**
</br>**So that** the input process is clearer and more structured

### Use Case 03: Validate Animal Name Length

**As a system**
</br>**I want** to ensure animal names are at least 2 characters
</br>**So that** meaningful names are provided

### Use Case 04: Validate Animal Species Against Allowed List

**As a system**
</br>**I want** to only accept predefined animal types
</br>**So that** data consistency is maintained

### Use Case 05: Validate Animal Colour Against Allowed List

**As a system**
</br>**I want** to only accept predefined colours
</br>**So that** colour data is standardized

### Use Case 06: Validate Limb Count as Non-Negative Integer

**As a system**
</br>**I want** to ensure limb count is a valid non-negative number
</br>**So that** only realistic numeric values are stored

### Use Case 07: Format and Standardize Animal Data Entry

**As a system**
</br>**I want** to automatically format user input to standard casing
</br>**So that** data is stored consistently regardless of how users type it

### Use Case 08: Remove Manual Counter Variable

**As a developer**
</br>**I want** to remove the redundant count variable
</br>**So that** the code is simpler and more maintainable

### Use Case 09: Build CSV String Using Join Method

**As a developer**
</br>**I want** to use Python's join method to construct CSV strings
</br>**So that** string concatenation is more Pythonic and efficient
